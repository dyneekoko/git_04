#html

- hyper text markup language
- HTML을 사용해서 웹페이지에 컨텐츠와 구조를 표시

## HTML basic

```
  <!DOCTYPE html> : 문서(웹페이지 문서들) 타입(종류) - 버전(HTML5 - W3C에서 확인)
  <html> : 웹 페이지 전체 영역
    <head> : 웹 페이지의 추가 정보(meta), 파일 임포트
      <title></title> : 꼭 넣어야함
    </head>

    <body> : 웹 페이지의 본문 영역 - 웹 페이지의 모든 콘텐츠를 표시하는 영역
    </body>
  </html>
```
